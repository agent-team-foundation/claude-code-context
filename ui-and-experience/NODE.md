# UI and Experience

This domain captures how Claude Code makes a complex agent runtime legible inside a terminal-first interface.

Subdomains:

- **[shell-and-input/](shell-and-input/NODE.md)** — Terminal shell composition, prompt entry, durable input history, keybindings, modal editing, voice, and workspace search overlays.
- **[startup-and-onboarding/](startup-and-onboarding/NODE.md)** — First-run/setup flows, welcome surfaces, spend acknowledgements, and low-priority recommendation/upsell bands.
- **[dialogs-and-approvals/](dialogs-and-approvals/NODE.md)** — Permission, plan, ask-user, export, diff, and focus-arbitrated dialog/overlay surfaces.
- **[feedback-and-notifications/](feedback-and-notifications/NODE.md)** — Feedback cues, footer stacks, out-of-band notifications, spinner hints, surveys, hook feedback, and idle-return summaries.
- **[background-and-teamwork/](background-and-teamwork/NODE.md)** — Background-task visibility, detail dialogs, teammate rendering, and companion-style auxiliary surfaces.
- **[transcript-and-history/](transcript-and-history/NODE.md)** — Transcript navigation, restore flows, resume browsing, and history-oriented interaction surfaces.
